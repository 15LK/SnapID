# SnapID — Post-Deploy Testing Checklist > Po nahrání `index.html` na GitHub → Vercel udělá deploy → otestuj v tomto pořadí. > Cíl: **20 minut**, žádný krok nepřeskočit. --- ## A) Před testem (1 minuta) - [ ] Vercel deploy je hotový (https://snap-id-five.vercel.app — zkontroluj v Vercel dashboardu zelený "Ready") - [ ] **Hard refresh** v prohlížeči: `Cmd+Shift+R` (Mac) — kvůli starému Service Worker cache - [ ] Pokud appka stále vypadá staře: DevTools → Application → Service Workers → **Unregister** → reload --- ## B) Test na MacBooku (Chrome / Safari) — 5 minut ### Vizuál - [ ] Hero karta má 4 rohové brackety (scan frame) ✅ - [ ] Grid 8 ikonek (Coins, Antiques, Jewelry, Plants, Art, Mushrooms, Insects, Rocks) ✅ - [ ] Sekce "How it works" se třemi kroky ✅ - [ ] Bottom nav: Home / Scan / Saved ✅ ### Theme toggle (🌙/☀️) - [ ] Klikni — barvy se přepnou - [ ] Reload stránky — theme zůstane uložený ### Upload tlačítkem - [ ] Klikni "Upload" → vybere se foto z disku - [ ] Preview se zobrazí - [ ] Klikni "Identify Now" → loading animace → výsledek - [ ] Result karta: jméno, kategorie, hodnota (zelená), origin, rarity (oranžová), confidence, demand bar, popis ### Sdílení jako obrázek (NOVÉ) - [ ] Klikni "📤 Share" na výsledku - [ ] Na Macu: stáhne se PNG soubor `snapid-.png` - [ ] Otevři PNG — měla by tam být tmavá karta s gradientem, jméno, hodnota, rarita, popis, branding "SnapID" ### Save & Search (NOVÉ) - [ ] Klikni "🔖 Save" → tlačítko zezelená "✅ Saved" - [ ] Jdi na záložku Saved → vidíš tam ten objekt - [ ] Search field nahoře → napiš část názvu → filtruje v reálném čase - [ ] Search "xyz" → vidíš empty state "No results for xyz" - [ ] Smaž search → znovu se objeví všechno --- ## C) Test na iPhonu (PWA + kamera) — 8 minut ### Otevři appku v Safari - [ ] https://snap-id-five.vercel.app - [ ] Pokud máš starou verzi: Safari → Settings → Safari → Clear Cookies/Data pro tento web → reload ### Camera tlačítko (KRITICKÉ — nové) - [ ] Klikni "Camera" - [ ] iOS se zeptá na povolení kamery → povol - [ ] Otevře se nativní camera UI (ne galerie) - [ ] Vyfoť něco → preview se objeví v appce - [ ] Identify → výsledek ### Upload z galerie - [ ] Klikni "Upload" - [ ] Otevře se galerie - [ ] Vyber foto → identify → výsledek ### PWA install - [ ] Safari → Share button → "Add to Home Screen" - [ ] Otevři z home screenu → běží jako standalone (bez Safari UI) - [ ] Theme color statusbaru — zatím #5B4BD4 fialová (TODO fix) ### Sdílení jako obrázek na iOS - [ ] Po výsledku klikni "📤 Share" - [ ] iOS Web Share API otevře native share sheet s PNG - [ ] Pošli si to do Photos / iMessage → ověř, že obrázek vypadá správně --- ## D) Test paywallu a Pro kódu — 3 minuty ### Free → Paywall - [ ] Otevři **incognito** okno (Mac) nebo Private (iOS) - [ ] Načti appku → vidíš "3 free scans" - [ ] Udělej 3 skeny → po 3. by se měl zobrazit paywall - [ ] Paywall obsahuje: - [ ] Tlačítko "Upgrade to Pro — $2.99/mo" → vede na Gumroad - [ ] Pole "Enter access code" - [ ] Aktivační tlačítko ### Aktivace Pro kódem - [ ] Zadej `SNAPID-PRO-2025` → klikni Activate - [ ] Zelený banner "✅ SnapID Pro activated" - [ ] Badge nahoře se změní na "⭐ Pro" - [ ] Můžeš dál skenovat bez limitu ### Test špatného kódu - [ ] Reset (smaž localStorage v DevTools) → zopakuj 3 skeny - [ ] Zadej `WRONG-CODE` → červený error "Invalid code" --- ## E) Smoke test backendu (1 minuta) - [ ] V terminálu / prohlížeči otevři: https://snapid-backend-production-0846.up.railway.app - [ ] Vidíš `{"status":"SnapID backend running"}` ✅ - [ ] V Vercel appce udělej skutečný sken — backend musí odpovědět do ~5 sekund --- ## F) Pokud něco selže ### Nový kód se neukáže (cache problém) 1. Vercel dashboard → ověř že deploy je "Ready", ne "Building" 2. Hard refresh: `Cmd+Shift+R` 3. DevTools → Application → Storage → "Clear site data" 4. iOS: Settings → Safari → Clear History and Website Data ### Identify vrací error 1. Otevři DevTools → Network → klikni na `/identify` request 2. Status 500 → backend issue → zkontroluj Railway logs 3. Status 401 → API klíč problém v Railway env vars 4. Status 0 / failed → CORS nebo network → zkontroluj Railway uptime ### Camera nefunguje na iOS - Safari musí být **HTTPS** (Vercel má, OK) - Permissions: iOS Settings → Safari → Camera → Allow - iOS 14.3+ je třeba (testuj na novějším iOS) ### Pro kód neaktivuje - Otevři DevTools Console → napiš `localStorage.getItem('snapid_pro')` - Mělo by vrátit `"true"` po úspěšné aktivaci - Pokud ne — chyba v JS, screenshotni mi konzoli --- ## G) Po úspěšném testu Pošli mi krátkou zprávu typu: - ✅ "Vše OK, jdeme dál" - ⚠️ "Funguje X a Y, ale Z je rozbité" + screenshot Pak budeme pokračovat **fixy z minulého reviewu** (image resize, sw.js verzování, theme ikona, manifest barva) — tentokrát už bez stresu deploye.
